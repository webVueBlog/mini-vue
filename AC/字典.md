字典

字典也是一种储存唯一值的数据结构，但它是以键值对的形式

js中的Map

常用操作：键值对的增删改查

const m = new Map();

// 增
m.set('a', 1);
m.set('b', 2);

// 删
m.delete('a');
// m.clear();

// 改,直接覆盖就行
m.set('b', 333)

// 查
const b = m.get('b');

console.log(m, b);
